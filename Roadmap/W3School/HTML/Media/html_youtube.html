Integrando Vídeos do YouTube em HTML

Incorporando um Vídeo do YouTube

Para exibir um vídeo do YouTube em uma página HTML, utilize a tag <iframe>. Defina o atributo src com a URL do vídeo e ajuste as dimensões do player com os atributos width e height.

Exemplo de código básico:

<iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY"></iframe>

Autoplay e Modo Mudo

Para que o vídeo seja reproduzido automaticamente, adicione autoplay=1 à URL. No entanto, alguns navegadores só permitem autoplay quando o vídeo está sem som. Use mute=1 para iniciar o vídeo automaticamente em modo mudo.

Exemplo de código com autoplay e mudo:

<iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=1&mute=1"></iframe>

Configuração de Playlist e Loop

Para configurar uma playlist, use uma lista de IDs separados por vírgula após playlist=. Adicione loop=1 para que o vídeo se repita continuamente.

Exemplo de código com loop e playlist:

<iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY?playlist=tgbNymZ7vqY&loop=1"></iframe>

Controle de Exibição

Você pode escolher se os controles de reprodução do vídeo serão exibidos. Use controls=0 para ocultar os controles, enquanto controls=1 (padrão) os exibe.

Exemplo de código sem controles:

<iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY?controls=0"></iframe>
