HTML Lists

As listas HTML permitem que os desenvolvedores web agrupem um conjunto de itens relacionados. Existem três tipos principais de listas: não ordenadas, ordenadas e de descrição.

Lista Não Ordenada

Uma lista não ordenada é iniciada com a tag <ul>, e cada item da lista começa com a tag <li>. Os itens são marcados com marcadores (círculos pretos pequenos) por padrão.

Exemplo:

  • Café
  • Chá
  • Leite

Lista Ordenada

Uma lista ordenada começa com a tag <ol>, e cada item também é definido com a tag <li>. Os itens são numerados por padrão.

Exemplo:

  1. Café
  2. Chá
  3. Leite

Lista de Descrição

HTML também suporta listas de descrição, que consistem em termos e suas descrições. A tag <dl> define a lista de descrição, <dt> define o termo (nome), e <dd> descreve cada termo.

Exemplo:

Café
- bebida quente e preta
Leite
- bebida fria e branca
