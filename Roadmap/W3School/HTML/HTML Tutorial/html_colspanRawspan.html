HTML Table Colspan & Rowspan

As tabelas em HTML podem apresentar células que se estendem por múltiplas colunas e/ou linhas, proporcionando uma melhor organização visual dos dados. O atributo colspan é utilizado para fazer uma célula abranger várias colunas. No exemplo abaixo, a célula de cabeçalho "Nome" se estende por duas colunas, ilustrando como podemos combinar células para melhorar a clareza.

Nome Idade
Jill Smith 43
Eve Jackson 57

Por outro lado, o atributo rowspan permite que uma célula se estenda por várias linhas. No exemplo a seguir, a célula "Telefone" abrange duas linhas, unindo informações relacionadas e garantindo que o layout permaneça claro e organizado.

Nome Jill
Telefone 555-1234
555-8745
