HTML Block and Inline Elements

Every HTML element has a default display value, depending on what type of element it is. The two most common display values are block and inline.

Block-level Elements

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

It also takes up the full width available, stretching out as far left and right as possible.

Two commonly used block elements are:

  • <p> - defines a paragraph
  • <div> - defines a division or section in an HTML document

Example of block elements:

Hello World

Hello World

Here are other block-level elements in HTML:

  • <address>
  • <article>
  • <aside>
  • <blockquote>
  • <video>

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary. For example, the <span> element is often used within a block-level element.

Example of inline elements:

This is a span element inside a paragraph.

Other inline elements include:

  • <a>
  • <abbr>
  • <acronym>
  • <b>
  • <var>

The <div> Element

The <div> element is a block-level container often used to group other HTML elements. It is commonly used with CSS to style sections of content.

Example of a styled <div> element:

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.

The <span> Element

The <span> element is an inline container used to style specific parts of the text. Like the <div> element, it has no required attributes but is often used with CSS to apply styles.

Example of a styled <span> element:

My mother has blue eyes and my father has dark green eyes.
