HTML Image Maps - Definições e Exemplos

Um mapa de imagens em HTML permite que você crie áreas clicáveis em uma imagem. Estas áreas são definidas com a tag <area>, enquanto o mapa de imagens é criado com a tag <map>.

Como Funciona?

O conceito por trás de um mapa de imagens é que você pode realizar diferentes ações, dependendo de onde clicar na imagem. A imagem é conectada ao mapa através do atributo usemap:

<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

O valor do usemap deve começar com #, seguido do nome do mapa, que é definido no atributo name da tag <map>.

Resumo

  • Use a tag <map> para definir um mapa de imagens.
  • A tag <area> define as áreas clicáveis dentro do mapa.
  • O atributo usemap da tag <img> é usado para associar a imagem ao mapa.
  • As áreas podem ser retangulares (rect), circulares (circle) ou poligonais (poly).
  • As áreas podem acionar eventos JavaScript usando o atributo onclick.
