Entendendo o Atributo class no HTML

O atributo class é utilizado para aplicar estilos a vários elementos de uma página, de forma eficiente e consistente. Além disso, pode ser manipulado com JavaScript para interatividade dinâmica.

Exemplo de Classe Única

O código abaixo atribui a classe city a três elementos div, permitindo que eles compartilhem o mesmo estilo:

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.

Aplicando Múltiplas Classes

É possível adicionar mais de uma classe a um elemento, permitindo que ele herde estilos de diferentes definições de classe. Neste exemplo, o primeiro título utiliza tanto a classe city quanto main:

London

Paris

Tokyo
