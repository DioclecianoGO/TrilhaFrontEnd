URLs (Uniform Resource Locators)

Uma URL, ou Localizador Uniforme de Recursos, é a forma como se referimos a um endereço da web. Ela pode ser constituída tanto por palavras, como em w3schools.com, quanto por um endereço de Protocolo de Internet (IP), como 192.68.20.50. Na navegação pela internet, os usuários geralmente utilizam nomes, pois são mais fáceis de memorizar do que sequências numéricas.

Função Principal

A função principal de uma URL é direcionar navegadores da web para solicitar páginas de servidores.

Sintaxe de uma URL

A sintaxe de uma URL é composta por diversos elementos, cada um com uma função específica. A estrutura básica inclui:

Estrutura de uma URL

scheme://prefix.domain:port/path/filename

Elementos da URL

  • scheme: define o tipo de serviço da Internet (ex: http ou https).
  • prefix: especifica um prefixo do domínio (ex: www).
  • domain: refere-se ao nome do domínio da Internet (ex: w3schools.com).
  • port: indica o número da porta no host (padrão para http é 80).
  • path: determina o caminho no servidor (se omitido, aponta para o diretório raiz do site).
  • filename: especifica o nome de um documento ou recurso.

Esquemas de URL Comuns

Esquema Abreviação Uso
http HyperText Transfer Protocol Páginas web comuns. Não criptografadas.
https Secure HyperText Transfer Protocol Páginas web seguras. Criptografadas.
ftp File Transfer Protocol Download ou upload de arquivos.
file Um arquivo no seu computador.

Codificação de URLs

As URLs devem ser enviadas pela Internet utilizando o conjunto de caracteres ASCII. Quando uma URL contém caracteres fora desse conjunto, é necessário convertê-los para um formato adequado. A codificação de URL substitui os caracteres não ASCII por um símbolo de porcentagem (%) seguido de dígitos hexadecimais.

Exemplo de Codificação de URL

Um espaço em uma URL é substituído por um sinal de mais (+) ou por %20.

Conclusão

A codificação assegura que a informação seja transmitida corretamente entre o navegador e o servidor, respeitando os padrões de codificação de caracteres, onde a codificação padrão em HTML5 é o UTF-8.
