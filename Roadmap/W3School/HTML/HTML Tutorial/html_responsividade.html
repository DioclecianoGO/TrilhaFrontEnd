Responsive Web Design - Examples and Explanations

Setting the Viewport

Para garantir que sua página web seja responsiva, use a seguinte meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Isso ajusta o viewport, permitindo que o navegador saiba como controlar as dimensões e o escalonamento da página.

Responsive Images

Imagens podem ser escaladas de forma responsiva para se ajustar ao tamanho do navegador. Aqui estão duas maneiras:

Using the width Property

Se você definir a largura da imagem como 100%, ela se ajustará ao tamanho da tela:

<img src="img_girl.jpg" style="width:100%;">

Using the max-width Property

Usando max-width: 100%, a imagem não será maior que seu tamanho original:

<img src="img_girl.jpg" style="max-width:100%;height:auto;">

Responsive Text

O tamanho do texto pode ser ajustado com a unidade vw (viewport width), que faz o texto escalar de acordo com o tamanho da janela.

<h1 style="font-size:10vw">Hello World</h1>

Using Media Queries

Media queries são usadas para aplicar diferentes estilos dependendo da largura da tela. No exemplo abaixo, três colunas são mostradas lado a lado em telas grandes, mas empilhadas em telas pequenas:

@media screen and (max-width: 800px) {
  .left, .main, .right {
    width: 100%;
  }
}

Responsive Frameworks

W3.CSS

O W3.CSS é um framework moderno que suporta design responsivo. Veja um exemplo simples usando W3.CSS:

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
