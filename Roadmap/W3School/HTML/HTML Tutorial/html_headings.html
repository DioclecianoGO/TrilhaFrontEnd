Como vimos, os headings vão de <h1> a <h6>, com <h1> sendo o mais importante. Cada nível de heading indica a estrutura hierárquica de um documento e deve ser usado para organizar o conteúdo da página.

Heading 1 Personalizado

Neste exemplo, o tamanho do <h1> foi alterado para 60 pixels usando o atributo style e a propriedade font-size do CSS. Isso permite ajustar visualmente o tamanho, mas mantendo a semântica correta do heading.

Note que os headings devem ser usados para estruturar o conteúdo, e não apenas para alterar o tamanho ou peso do texto. Motores de busca, como o Google, usam esses elementos para entender a organização da página.
