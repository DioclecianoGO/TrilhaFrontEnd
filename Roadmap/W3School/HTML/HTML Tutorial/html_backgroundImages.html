Exemplos de Imagens de Fundo com HTML e CSS

Imagem de Fundo em Elementos HTML

Este parágrafo tem uma imagem de fundo aplicada diretamente no atributo style.

Imagem de Fundo Definida no CSS

Neste exemplo, a imagem de fundo foi definida no CSS usando a propriedade background-image.

Imagem de Fundo para a Página Inteira

Aqui, aplicamos uma imagem de fundo em todo o corpo da página, utilizando o elemento body e as propriedades CSS adequadas.

Imagem de Fundo Sem Repetição

Quando a imagem de fundo não é repetida, ela aparece uma única vez. Isso é feito com a propriedade background-repeat: no-repeat.

Imagem de Fundo Com Cobertura Total

A imagem de fundo cobre completamente o elemento, sem distorções. Usamos background-size: cover para garantir isso.

Imagem de Fundo Esticada

Neste exemplo, a imagem de fundo é esticada para preencher todo o elemento, usando background-size: 100% 100%.
