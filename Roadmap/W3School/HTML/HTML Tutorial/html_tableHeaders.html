HTML Table Headers

Os cabeçalhos em tabelas HTML organizam o conteúdo das colunas ou linhas e ajudam a identificar o que cada dado representa. Eles são definidos pela tag <th>, que estiliza os conteúdos de cabeçalho em negrito e centralizados, destacando-os das células normais, que usam a tag <td>.

Exemplo básico de cabeçalhos

O exemplo abaixo demonstra o uso de cabeçalhos para identificar cada coluna da tabela:

Exemplo de Cabeçalhos Horizontais
Nome Sobrenome Idade
Jill Smith 50
Eve Jackson 94

Note que os cabeçalhos <th> da primeira linha descrevem o conteúdo das colunas subsequentes. Por padrão, eles são centralizados e exibidos em negrito.

Cabeçalhos Verticais

Além dos cabeçalhos horizontais, é possível organizar os cabeçalhos de forma vertical, usando a tag <th> na primeira célula de cada linha:

Exemplo de Cabeçalhos Verticais
Nome Jill Eve
Sobrenome Smith Jackson
Idade 50 94

Neste caso, as células <th> rotulam os dados que estão em cada linha, em vez das colunas.

Alinhamento dos Cabeçalhos

Embora os cabeçalhos sejam centralizados por padrão, você pode alterar o alinhamento com CSS. No exemplo abaixo, os cabeçalhos são alinhados à esquerda:

Cabeçalhos Alinhados à Esquerda
Nome Sobrenome Idade
Jill Smith 50
Eve Jackson 94

Cabeçalhos Abrangendo Múltiplas Colunas

Em alguns casos, você pode querer que um cabeçalho se estenda por várias colunas. Isso é possível com o uso do atributo colspan:

Cabeçalhos com Colspan
Nome Completo Idade
Jill Smith 50
Eve Jackson 94

O atributo colspan="2" faz com que o cabeçalho "Nome Completo" abranja as duas primeiras colunas.

Legenda da Tabela

Você pode adicionar uma legenda à tabela com a tag <caption>. Ela funciona como um título para a tabela, facilitando a identificação do conteúdo apresentado.
