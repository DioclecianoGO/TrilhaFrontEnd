Exemplos de Formatação de Texto em HTML

Texto em Negrito

O elemento <b> exibe o texto em negrito, sem importância adicional. Exemplo:

Este texto está em negrito

Texto com Importância

O elemento <strong> exibe o texto em negrito com importância adicional. Exemplo:

Este texto é importante!

Texto em Itálico

O elemento <i> exibe o texto em itálico. Exemplo:

Este texto está em itálico

Texto Enfatizado

O elemento <em> exibe o texto em itálico e com ênfase adicional. Exemplo:

Este texto é enfatizado

Texto Menor

O elemento <small> exibe o texto com um tamanho menor. Exemplo:

Este é um texto menor.

Texto Marcado

O elemento <mark> destaca o texto como se fosse marcado. Exemplo:

Não se esqueça de comprar leite hoje.

Texto Deletado

O elemento <del> exibe o texto riscado, indicando que foi deletado. Exemplo:

Minha cor favorita é azul vermelha.

Texto Inserido

O elemento <ins> sublinha o texto inserido. Exemplo:

Minha cor favorita é azul vermelha.

Texto Subscrito

O elemento <sub> exibe o texto como subscrito. Exemplo:

H2O é a fórmula da água.

Texto Sobrescrito

O elemento <sup> exibe o texto como sobrescrito. Exemplo:

Esta é uma nota1.
