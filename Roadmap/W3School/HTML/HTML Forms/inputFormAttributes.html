Atributos de Formulário HTML

form

O atributo form especifica a qual formulário o elemento <input> pertence, correspondendo ao id de um formulário <form> existente.

<form id="form1">
  <input type="text" name="nome">
</form>
<input type="text" name="sobrenome" form="form1">

formaction

O atributo formaction define a URL para onde os dados do formulário serão enviados, sobrescrevendo o atributo action do <form>.

<form action="/acao_padrao.php">
  <input type="submit" value="Enviar">
  <input type="submit" formaction="/outra_acao.php" value="Enviar para outra ação">
</form>

formenctype

O atributo formenctype define o tipo de codificação dos dados do formulário ao serem enviados. Aplica-se somente ao método post.

<form action="/upload" method="post">
  <input type="submit" formenctype="multipart/form-data" value="Enviar como multipart">
</form>

formmethod

O atributo formmethod define o método HTTP para enviar os dados do formulário, podendo ser GET ou POST.

<form action="/enviar" method="get">
  <input type="submit" formmethod="post" value="Enviar como POST">
</form>

formtarget

O atributo formtarget especifica onde a resposta do formulário será exibida, como em uma nova aba ou janela.

<form action="/resultado">
  <input type="submit" formtarget="_blank" value="Abrir em nova aba">
</form>

formnovalidate

O atributo formnovalidate indica que o formulário não deve ser validado ao ser submetido.

<form action="/envio">
  <input type="email" required>
  <input type="submit" formnovalidate value="Enviar sem validação">
</form>
