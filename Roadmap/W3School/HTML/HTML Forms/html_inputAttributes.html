Atributos de Entrada em HTML

Atributo: value

O atributo value define um valor inicial em um campo de entrada.

Atributo: readonly

O atributo readonly permite que o usuário visualize o conteúdo sem modificá-lo.

Atributo: disabled

O atributo disabled desativa o campo, impedindo interação e envio.

Atributo: size

O atributo size controla a largura do campo de entrada em número de caracteres.

Atributo: maxlength

O atributo maxlength limita o número máximo de caracteres permitidos.

Atributos: min e max

Os atributos min e max definem valores mínimo e máximo para campos numéricos e de data.

Atributo: multiple

O atributo multiple permite selecionar múltiplos valores, como em um campo de upload de arquivos.

Atributo: pattern

O atributo pattern define uma expressão regular para validar o formato da entrada.

Atributo: placeholder

O atributo placeholder fornece uma dica visual sobre o valor esperado no campo.

Atributo: required

O atributo required exige que o campo seja preenchido antes do envio do formulário.

Atributo: step

O atributo step permite definir intervalos específicos para campos numéricos ou de faixa.

Atributo: autofocus

O atributo autofocus foca automaticamente em um campo ao carregar a página.

Atributos: height e width

Os atributos height e width configuram o tamanho de imagens em campos de entrada.

Atributo: list e datalist

O atributo list, junto ao elemento datalist, cria uma lista de sugestões para o usuário.

Atributo: autocomplete

O atributo autocomplete controla se o navegador sugere valores previamente inseridos.
