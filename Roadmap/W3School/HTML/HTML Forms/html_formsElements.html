Explicação dos Elementos de Formulário em HTML

Elemento <input>

O elemento <input> é um dos mais versáteis para formulários, permitindo diferentes tipos de entrada de usuário por meio do atributo type.

<input type="text" id="name" name="name">

Elemento <label>

O elemento <label> é usado para definir um rótulo para outros elementos de formulário. Ele se associa a um <input> usando o atributo for que corresponde ao id do input.

<label for="email">E-mail:</label><input type="email" id="email" name="email">

Elemento <select>

O elemento <select> cria uma lista suspensa. Cada opção é definida com a tag <option>.

<select id="car" name="car">...</select>

Elemento <textarea>

O elemento <textarea> define uma entrada de texto em várias linhas, útil para comentários ou feedbacks mais longos.

<textarea id="message" name="message" rows="4" cols="50"></textarea>

Elemento <button>

O elemento <button> define um botão clicável. Sempre especifique o atributo type para um comportamento consistente.

<button type="button" onclick="alert('Botão clicado!')">Clique Aqui!</button>

Elementos <fieldset> e <legend>

O elemento <fieldset> agrupa elementos relacionados, e <legend> fornece um título para o grupo.

Informações Pessoais

<fieldset><legend>...</fieldset>

Elemento <datalist>

O elemento <datalist> fornece uma lista de opções pré-definidas para um campo de entrada, criando uma experiência de combo box.

<input list="browsers" id="browser"><datalist id="browsers">...</datalist>

Elemento <output>

O elemento <output> representa o resultado de um cálculo, que pode ser atualizado dinamicamente com base nas entradas.

+ = 30

<output name="result" for="a b"></output>
