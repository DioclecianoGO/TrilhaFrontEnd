CSS: Trabalhando com Cores

Nomes de Cores Predefinidos

O CSS oferece 140 nomes de cores predefinidos que podem ser utilizados diretamente.

Exemplo de cor predefinida: Tomato.

Cores de Fundo e de Texto

Podemos definir a cor do fundo com background-color e a cor do texto com color.

Texto com cor "DodgerBlue".

Fundo com cor "LightGray".

Cor das Bordas

A cor das bordas pode ser definida com border-color, aplicando estilos diferentes em cada lado.

Borda com cores diferentes: topo (Tomato), direita (DodgerBlue), inferior (Violet), esquerda (Gray).

Valores RGB e RGBA

O modelo RGB define a cor com valores de Vermelho, Verde e Azul. RGBA adiciona transparência ao permitir um valor de 0 a 1.

Texto com RGB: Vermelho puro (rgb(255, 0, 0)).

Fundo com RGBA: Verde semitransparente (rgba(0, 128, 0, 0.5)).

Valores HEX

A notação hexadecimal define cores com #rrggbb ou #rgb para cores abreviadas.

Texto com HEX: Vermelho alaranjado (#ff5733).

Texto com HEX abreviado: Verde claro (#0f0).

Valores HSL e HSLA

O modelo HSL usa matiz, saturação e luminosidade para definir a cor. HSLA adiciona transparência com o canal Alpha.

Texto com HSL: Azul puro (hsl(240, 100%, 50%)).

Fundo com HSLA: Verde escuro semitransparente (hsla(120, 100%, 25%, 0.5)).
