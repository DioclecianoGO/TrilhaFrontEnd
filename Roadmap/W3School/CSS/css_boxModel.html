Modelo de Caixa (Box Model) no CSS

Componentes do Box Model

O Modelo de Caixa é composto de quatro partes principais:

  • Conteúdo: onde os textos e imagens aparecem.
  • Padding: espaço transparente ao redor do conteúdo.
  • Borda: circunda o conteúdo e o padding, destacando a área interna.
  • Margem: espaço transparente entre a borda do elemento e os elementos vizinhos.

Cálculo da Largura Total

Para calcular a largura total de um elemento, somamos:

  • Conteúdo: largura definida (ex.: 200px).
  • Padding: 20px de cada lado (total de 40px).
  • Borda: 5px de cada lado (total de 10px).

No exemplo acima, a largura total da caixa é: 250px (200px + 40px + 10px).

Propriedade box-sizing: border-box

A propriedade box-sizing: border-box inclui padding e bordas dentro das dimensões especificadas, mantendo a largura e altura definidas:

Este é um exemplo com box-sizing: border-box, onde a largura total permanece 200px, incluindo o padding e a borda.
