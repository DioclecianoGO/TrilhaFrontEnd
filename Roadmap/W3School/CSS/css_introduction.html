Introdução ao CSS - Cascading Style Sheets

Exemplo prático de estilos CSS aplicados em HTML.

O que é o CSS?

CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar páginas da web. Ela controla a aparência dos documentos HTML, centralizando a definição de estilo em arquivos externos. Essa abordagem facilita a manutenção e evita a repetição de estilos em cada página individual.

Por que usar CSS?

CSS permite organização e consistência ao estilizar sites. Um arquivo .css externo oferece uniformidade, permitindo que alterações feitas neste arquivo impactem todas as páginas que o referenciam. Ele também permite responsividade, adaptando o design para diferentes dispositivos e tamanhos de tela.

Syntax

Uma regra CSS é composta por um seletor e um bloco de declarações. O seletor indica o elemento HTML a ser estilizado, e o bloco de declarações define as propriedades CSS. Exemplo:

p { color: red; text-align: center; }

Este parágrafo está centralizado e em vermelho, conforme a regra CSS acima.

Selectors

CSS oferece uma variedade de seletores para identificar e aplicar estilos a elementos HTML. Aqui estão alguns exemplos:

  • Seletor de elemento: aplica estilo a todos os elementos de um tipo específico, como p.
  • Seletor de ID: estiliza um elemento único usando #id. Exemplo: Estilo único.
  • Seletor de classe: estiliza elementos com uma classe específica, usando .classe. Exemplo: Texto com classe highlight.
  • Seletor universal: estiliza todos os elementos usando *.
  • Seletor de agrupamento: aplica o mesmo estilo a vários elementos separados por vírgula.

Inserindo CSS

Há três formas principais de adicionar CSS:

  • CSS Externo: Referenciado por meio de <link> em cada página. Veja styles.css neste exemplo.
  • CSS Interno: Definido dentro da tag <style> no <head>. Exemplo nesta página.
  • CSS Inline: Aplicado diretamente ao elemento HTML. Exemplo: Texto com CSS inline em verde.

Comentários

Comentários em CSS são adicionados com /* comentário */ e ajudam na documentação do código.

/* Este é um comentário em CSS */

Em HTML, comentários são feitos assim: <!-- comentário -->.
