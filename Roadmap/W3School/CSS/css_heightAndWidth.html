Propriedades de Altura e Largura em CSS

As propriedades height, width, e max-width são essenciais no CSS para controlar as dimensões dos elementos. Aqui, exploraremos como essas propriedades funcionam e exemplos práticos.

auto (Padrão)

Com a propriedade auto, o navegador ajusta automaticamente a largura e a altura com base no conteúdo.

Este box tem altura e largura ajustadas automaticamente.

Valores Fixos

Quando usamos unidades como px, definimos dimensões fixas.

Exemplo de Box com largura e altura fixas:

Box com largura de 150px e altura de 100px.

Porcentagem

Usando %, a largura é calculada em relação à largura do elemento pai.

Exemplo de Box com largura em 50%:

Box com largura de 50% do contêiner.

max-width

A propriedade max-width define o limite máximo de largura do elemento.

Exemplo de Box com max-width de 200px:

Box com largura máxima de 200px, ajustando-se conforme o tamanho do contêiner.

box-sizing: border-box

Usando box-sizing: border-box, o padding é incluído na largura total do elemento.

Exemplo de Box com box-sizing:

Box com box-sizing border-box, incluindo o padding dentro da largura de 200px.

Conclusão

As propriedades height, width e max-width são ferramentas poderosas no CSS, essenciais para controle de layout e criação de designs responsivos. A interação entre essas propriedades e o uso de box-sizing são importantes para garantir um design funcional e visualmente agradável.
