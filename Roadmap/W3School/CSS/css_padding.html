Propriedades de Padding no CSS

Em CSS, o padding é o espaço interno entre o conteúdo de um elemento e suas bordas, conhecido como o "respiro" do conteúdo.

Exemplo de Padding Básico

Neste exemplo, aplicamos um padding uniforme em todos os lados do elemento:

Este elemento possui padding de 20px em todos os lados.

.padding-exemplo {
    padding: 20px;
}

Padding Individual para Cada Lado

É possível definir o padding de cada lado do elemento separadamente:

Este elemento possui padding de 10px no topo, 20px à direita, 30px embaixo e 40px à esquerda.

.padding-individual {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
}

Sintaxe Abreviada para Padding

Com a sintaxe abreviada, é possível configurar todos os lados com uma linha de código:

Padding com valores diferentes para cada lado: 10px (topo), 20px (direita), 30px (inferior), 40px (esquerda).

.padding-abreviado {
    padding: 10px 20px 30px 40px;
}

Box-sizing: Border-box

Usar box-sizing: border-box; evita que o padding aumente a largura total do elemento:

Este elemento possui box-sizing: border-box;, o que permite que o padding seja incluído dentro da largura especificada (250px).

.box-sizing-exemplo {
    padding: 20px;
    box-sizing: border-box;
}
