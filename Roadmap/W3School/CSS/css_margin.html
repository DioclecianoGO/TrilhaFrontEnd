Propriedade Margin em CSS

Composição e Sintaxe

A propriedade margin é usada para adicionar espaços ao redor dos elementos. Ela pode ser aplicada individualmente para cada lado ou de forma abreviada para definir as margens de todos os lados de uma só vez.

Estrutura e Funcionamento das Margens

As margens podem ser configuradas individualmente para cada lado do elemento:

margin-top: 20px;
margin-right: 30px;
margin-bottom: 40px;
margin-left: 50px;

Unidades de Medida

Você pode definir as margens com diferentes unidades de medida:

Margin com valor fixo (px)
Margin com valor relativo (em)
Margin com valor percentual (%)
Margin auto para centralização

Sintaxe Abreviada

A propriedade margin pode ser escrita de forma abreviada para simplificar o código. Veja os exemplos abaixo:

margin: 20px 30px 40px 50px;
margin: 20px 30px 40px;
margin: 20px 30px;
margin: 20px;

Valor inherit

Utilizando margin: inherit, o elemento herda as margens do elemento pai:

Este box tem margens de 30px

Colapso de Margens

Quando duas margens verticais se encontram, elas se "fundem" em uma única. O valor final será o maior entre as duas margens adjacentes.
