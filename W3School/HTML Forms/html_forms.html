HTML Forms

The <form> Element

O elemento <form> é usado para criar um formulário que coleta dados do usuário.

Sintaxe básica:

<form>
  <!-- elementos do formulário -->
</form>

The <input> Element

O elemento <input> pode ser exibido de diferentes formas com base no valor do atributo type.

Campos de Texto:

O <input type="text"> define um campo de entrada de uma única linha.


Botões de Rádio:

O <input type="radio"> permite que o usuário escolha uma opção de várias disponíveis.

Checkboxes:

O <input type="checkbox"> permite que o usuário selecione zero ou mais opções.

The Name Attribute

Cada campo de entrada no formulário deve ter um atributo name para que seus valores sejam enviados ao servidor.

Se o atributo name for omitido, o valor não será submetido.
