Exemplos e Explicações de Inputs HTML

Texto (type="text")

Entrada básica para capturar texto livre.

Senha (type="password")

Oculta o texto digitado, ideal para capturar senhas.

Email (type="email")

Valida automaticamente o formato do e-mail.

Telefone (type="tel")

Permite a entrada de números de telefone.

URL (type="url")

Espera um endereço de site válido.

Número (type="number")

Permite apenas valores numéricos, com controles de valor mínimo e máximo.

Data (type="date")

Exibe um seletor de data.

Cor (type="color")

Permite escolher uma cor usando um seletor de cores.

Arquivo (type="file")

Permite o upload de arquivos.

Checkbox (type="checkbox")

Permite múltiplas seleções.

Radio (type="radio")

Permite apenas uma seleção em um grupo.

Faixa (type="range")

Entrada deslizante, ideal para valores de intervalo.

Botões de Submissão e Reset

Botão de envio para processar os dados e botão de reset para limpar o formulário.
