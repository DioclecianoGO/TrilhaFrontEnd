Exemplos de Atributos de Formulário em HTML

Atributo Action

O atributo action define o destino dos dados quando o formulário é submetido. Geralmente, é o caminho de um arquivo que processa os dados no servidor.

Exemplo de uso: Neste exemplo, os dados serão enviados para action_page.php:

<form action="/action_page.php">
  <label for="fname">Nome:</label>
  <input type="text" id="fname" name="fname">
  <input type="submit" value="Enviar">
</form>

Atributo Target

O atributo target especifica onde a resposta será exibida após o envio do formulário.

Principais valores:

  • _blank: abre a resposta em uma nova aba ou janela.
  • _self: abre a resposta na mesma janela (padrão).

Exemplo de uso: Neste exemplo, a resposta abre em uma nova aba:

<form action="/action_page.php" target="_blank">
  <label for="lname">Sobrenome:</label>
  <input type="text" id="lname" name="lname">
  <input type="submit" value="Enviar">
</form>

Atributo Method

O atributo method define o método HTTP para o envio dos dados, podendo ser get ou post.

GET: Envia dados anexados à URL. Não deve ser usado para informações sensíveis.

POST: Envia dados no corpo da requisição, recomendado para dados confidenciais.

Exemplo de uso: Envio usando o método POST:

<form action="/action_page.php" method="post">
  <label for="email">Email:</label>
  <input type="text" id="email" name="email">
  <input type="submit" value="Enviar">
</form>

Atributo Autocomplete

O atributo autocomplete especifica se o navegador deve sugerir preenchimentos automáticos para campos do formulário.

Exemplo de uso: O formulário abaixo possui preenchimento automático ativado:

<form action="/action_page.php" autocomplete="on">
  <label for="username">Usuário:</label>
  <input type="text" id="username" name="username">
  <input type="submit" value="Enviar">
</form>

Atributo Novalidate

O atributo novalidate desativa a validação do navegador ao enviar o formulário.

Exemplo de uso: O formulário abaixo não será validado ao enviar:

<form action="/action_page.php" novalidate>
  <label for="phone">Telefone:</label>
  <input type="text" id="phone" name="phone" required>
  <input type="submit" value="Enviar">
</form>
