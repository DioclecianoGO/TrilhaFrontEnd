Usando o Elemento de Vídeo no HTML

Introdução ao Elemento <video>

O elemento <video> permite a incorporação de vídeos diretamente em páginas web. Com ele, é possível exibir vídeos com controles personalizados para o usuário, como play, pause e ajuste de volume, através do atributo controls.

Sintaxe do Elemento <video>

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Seu navegador não suporta o elemento de vídeo.
</video>

Essa estrutura permite que o navegador escolha o primeiro formato de vídeo suportado entre as opções definidas com a tag <source>.

Reprodução Automática e Silenciamento

Para iniciar o vídeo automaticamente, use o atributo autoplay. No entanto, navegadores modernos permitem autoplay somente em vídeos sem som, o que pode ser resolvido adicionando o atributo muted junto ao autoplay.

Formatos de Vídeo Suportados

Os formatos de vídeo compatíveis com HTML são:

  • MP4 (video/mp4) - Suportado por todos os navegadores e recomendado para uso geral.
  • WebM (video/webm) - Formato eficiente para navegadores modernos.
  • Ogg (video/ogg) - Suportado por vários navegadores, mas não pelo Safari.

Esses formatos podem ser utilizados dentro da tag <video> com o elemento <source> para que o navegador selecione o primeiro formato disponível.

Manipulação de Vídeo com o DOM HTML

O DOM HTML permite controlar vídeos com métodos, propriedades e eventos. Por exemplo, é possível iniciar ou pausar o vídeo programaticamente, ajustar o volume e monitorar eventos como o início e a pausa do vídeo.
