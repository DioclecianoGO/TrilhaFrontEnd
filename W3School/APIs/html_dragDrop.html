HTML Drag and Drop API

Introdução à Drag and Drop API

A API de arrastar e soltar permite mover elementos na página, criando uma experiência interativa. Este recurso é amplamente utilizado em interfaces modernas, como gerenciadores de arquivos e organizadores de conteúdo, onde os usuários podem reposicionar itens facilmente.

Definir Elemento como Arrastável

Para tornar um elemento arrastável, definimos o atributo draggable="true". Isso indica que o elemento pode ser movido pelo usuário.

<img id="drag1" src="img_logo.gif" draggable="true">

Eventos de Arraste e Soltura

A API utiliza eventos para controlar o fluxo de arrastar e soltar:

  • ondragstart: Inicia o arraste e define os dados a serem transferidos.
  • ondragover: Permite que o elemento seja solto em uma área específica chamando preventDefault().
  • ondrop: Conclui o processo, pegando o elemento arrastado e adicionando-o ao novo local de destino.

Explicação do Código

  1. allowDrop(event): Permite que o elemento seja solto chamando event.preventDefault().
  2. drag(event): Define o id do elemento arrastado com dataTransfer.setData() para que ele possa ser recuperado no evento de soltura.
  3. drop(event): Recupera o id do elemento arrastado com dataTransfer.getData() e o adiciona ao novo local de destino usando appendChild().

Aplicações e Benefícios

A funcionalidade de arrastar e soltar facilita a manipulação de elementos visualmente. Alguns usos comuns incluem:

  • Organização de listas e itens em aplicativos de planejamento.
  • Interfaces de gerenciadores de arquivos que permitem arrastar itens para pastas.
  • Editores de layout onde os usuários podem rearranjar componentes.
