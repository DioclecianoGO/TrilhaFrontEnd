Imagens em HTML

Imagens podem melhorar o design e a aparência de uma página web. Abaixo estão as explicações e exemplos práticos sobre como usar imagens em HTML.

Tag <img> no HTML

A tag <img> é usada para incorporar uma imagem em uma página HTML. Ela é uma tag auto-contida e não requer uma tag de fechamento. Duas de suas propriedades mais importantes são:

  • src: Especifica o caminho para a imagem.
  • alt: Fornece um texto alternativo caso a imagem não possa ser exibida.

Exemplo de uso básico

<img src="pic_trulli.jpg" alt="Italian Trulli">

Atributo src

O atributo src é usado para especificar o caminho para a imagem. O navegador carrega a imagem a partir do servidor e a insere na página quando ela é carregada.

Exemplo de src correto

<img src="img_chania.jpg" alt="Flowers in Chania">

Atributo alt

O atributo alt fornece um texto alternativo caso a imagem não seja carregada ou em situações de acessibilidade, como no uso de leitores de tela.

Exemplo com erro no src, exibindo o alt

<img src="imagem_incorreta.jpg" alt="Flowers in Chania">

Definindo Tamanho da Imagem

Você pode definir o tamanho de uma imagem usando os atributos width e height ou através do CSS com o atributo style. Isso ajuda a controlar o layout da página e melhorar a experiência do usuário.

Exemplo usando width e height

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Exemplo usando CSS com o atributo style

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

Imagens em Diretórios Diferentes

Se a imagem estiver em uma pasta diferente, você deve especificar o caminho relativo ou absoluto.

Exemplo de imagem em subpasta

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Imagens Externas

Imagens podem ser referenciadas de outros servidores usando um caminho absoluto. Tenha cuidado com o uso de imagens externas, pois elas podem estar sujeitas a direitos autorais ou serem removidas pelo servidor original.

Exemplo com URL absoluto

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Imagens Animadas

O HTML também permite a inserção de GIFs animados.

Exemplo de GIF animado

<img src="programming.gif" alt="Computer Man" style="width:48px;height:48px;">

Imagem como Link

Você pode usar uma imagem como link, colocando a tag <img> dentro da tag <a>.

Exemplo de imagem clicável

<a href="https://www.w3schools.com">
<img src="https://www.w3schools.com/html/img_chania.jpg" alt="Flowers in Chania" style="width:100px;height:100px;">
</a>

Imagem Flutuante (Float)

A propriedade CSS float permite que uma imagem flutue ao lado do texto.

Exemplo de imagem flutuando à direita

<p><img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;">Texto ao redor da imagem.</p>

Texto ao redor da imagem. Ela está flutuando à direita do parágrafo.

Formatos Comuns de Imagem

Abaixo estão os formatos de imagem mais comuns suportados pelos navegadores:

  • APNG - Animated Portable Network Graphics (.apng)
  • GIF - Graphics Interchange Format (.gif)
  • ICO - Microsoft Icon (.ico)
  • JPEG - Joint Photographic Expert Group (.jpg, .jpeg)
  • PNG - Portable Network Graphics (.png)
  • SVG - Scalable Vector Graphics (.svg)
