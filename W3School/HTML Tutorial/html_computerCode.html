HTML Computer Code Elements

Elemento <kbd> para Entrada de Teclado

O elemento <kbd> é usado para definir entradas de teclado. O texto dentro deste elemento é exibido com uma fonte monoespaçada, indicando que o usuário deve apertar as teclas mencionadas.

Código:

<p>Save the document by pressing <kbd>Ctrl + S</kbd></p>

Resultado:

Save the document by pressing Ctrl + S

Elemento <samp> para Saída de Programa

O elemento <samp> é utilizado para mostrar a saída de um programa de computador, exibido em fonte monoespaçada. Ideal para mostrar mensagens geradas por um sistema.

Código:

<p>Message from my computer:</p>
<p><samp>File not found.<br>Press F1 to continue</samp></p>

Resultado:

Message from my computer:

File not found.
Press F1 to continue

Elemento <code> para Código de Computador

O elemento <code> é usado para representar um trecho de código de computador. Contudo, ele não preserva quebras de linha ou espaços extras.

Código:

<code>
x = 5;
y = 6;
z = x + y;
</code>

Resultado:

x = 5; y = 6; z = x + y;

Para preservar a formatação, utilize <pre> em conjunto com o elemento <code>:

Código:

<pre><code>
x = 5;
y = 6;
z = x + y;
</code></pre>

Resultado:

x = 5;
y = 6;
z = x + y;

Elemento <var> para Variáveis

O elemento <var> é utilizado para definir variáveis em expressões matemáticas ou programação, exibido geralmente em itálico.

Código:

<p>The area of a triangle is: 1/2 x <var>b</var> x <var>h</var>, where <var>b</var> is the base, and <var>h</var> is the vertical height.</p>

Resultado:

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.
