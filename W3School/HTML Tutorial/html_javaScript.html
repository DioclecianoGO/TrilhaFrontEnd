HTML e JavaScript

JavaScript é uma linguagem de programação que torna as páginas HTML mais dinâmicas e interativas. Ele pode ser utilizado para manipulação de imagens, validação de formulários e alterações dinâmicas de conteúdo.

A Tag <script>

O elemento <script> é usado para definir um script do lado do cliente, podendo conter instruções JavaScript ou apontar para um arquivo externo através do atributo src.

Para selecionar um elemento HTML, JavaScript frequentemente utiliza o método document.getElementById(). Por exemplo:

document.getElementById("demo").innerHTML = "Hello JavaScript!";

O que o JavaScript pode fazer

  • Alterar Conteúdo: Modifica o texto de um elemento HTML.
  • Alterar Estilos: Muda propriedades de estilo, como tamanho e cor da fonte.
  • Alterar Atributos: Atualiza atributos de elementos, como a fonte de uma imagem.

Tag <noscript>

O elemento <noscript> é utilizado para fornecer conteúdo alternativo para usuários que têm scripts desativados em seus navegadores ou que utilizam navegadores que não suportam scripts. Por exemplo:

<noscript>Sorry, your browser does not support JavaScript!</noscript>

Exemplo de JavaScript

O código abaixo demonstra como o JavaScript pode ser utilizado:


<script>
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
<noscript>Sorry, your browser does not support JavaScript!</noscript>
