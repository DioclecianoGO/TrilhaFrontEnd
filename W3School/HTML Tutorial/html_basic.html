<!DOCTYPE html>

A declaração de

<!DOCTYPE html> Representa o tipo de documento, ajuda o navegador apresentar páginas corretamente. Ele só deve aparecer uma vez, no topo da página (antes de quaisquer tags HTML).

<html>

O documento HTML inicia e termina com a tag

<html>

<body>

Mas a parte visivel do documento html, está no

<body>

Heading <h1>

Os títulos HTML são definidos com o

<h1> para <h6> De forma descrescente <h1> define o título mais importante em direção ao <h6> menos importante.

<h1>This is heading 1</h1> <h2>This is heading 1</h2> <h3>This is heading 1</h3> <h4>This is heading 1</h4>

Parágrafos <p>

O parágrafo é definido com a tag

<p>

Links <a>

O link é definido com a tag

<a> O destino do link é especificado no atributo href. Os atributos são usados para fornecer informações adicionais sobre elementos HTML.

Exemplo

<a href="https://www.w3schools.com">This is a link</a>

Imagens <img>

As imagens são definido com a tag

<img>

Exemplo

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
