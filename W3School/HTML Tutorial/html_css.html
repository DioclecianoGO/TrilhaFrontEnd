CSS Externo

Para usar CSS externo, é necessário vincular o arquivo com a tag <link> no <head>.

Exemplo de uso:

        <link rel="stylesheet" href="styles.css">

Arquivo Externo (styles.css)

O arquivo styles.css pode conter algo como:

        body {
            background-color: powderblue;
        }
        h1 {
            color: blue;
        }
        p {
            color: red;
        }
