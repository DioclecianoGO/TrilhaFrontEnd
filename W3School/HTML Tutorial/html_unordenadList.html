HTML Unordered Lists

As listas não ordenadas em HTML são definidas pela tag <ul>, que cria uma estrutura de itens apresentados com marcadores. Cada item da lista é iniciado com a tag <li>, e, por padrão, os itens são exibidos com pequenos círculos pretos como marcadores. Essa funcionalidade permite que desenvolvedores web organizem informações de maneira clara e visualmente acessível.

Uma característica importante das listas não ordenadas é a possibilidade de personalizar o estilo dos marcadores utilizando a propriedade CSS list-style-type. Essa propriedade permite que os desenvolvedores escolham entre várias opções de estilo, como círculos, quadrados ou até mesmo a remoção total dos marcadores. Com essa flexibilidade, é possível adaptar a aparência das listas de acordo com o design desejado para uma página.

Outra funcionalidade interessante é a capacidade de aninhar listas. Isso significa que um item dentro de uma lista pode conter outra lista, permitindo a criação de hierarquias e a organização de informações de forma mais estruturada. Essa prática é especialmente útil para categorizar dados relacionados, tornando a apresentação mais compreensível.

Além disso, as listas podem ser estilizadas com CSS para serem exibidas horizontalmente, o que é uma abordagem comum em menus de navegação em sites. Ao remover os marcadores e aplicar flutuação aos itens, é possível criar um layout de menu que melhora a experiência do usuário. A combinação de HTML e CSS proporciona uma maneira eficaz de apresentar informações de forma organizada e esteticamente agradável, permitindo que os desenvolvedores criem interfaces interativas e funcionais.

Exemplos de Listas Não Ordenadas

Exemplo Básico

Uma lista não ordenada simples com itens:

        <ul>
            <li>Café</li>
            <li>Chá</li>
            <li>Leite</li>
        </ul>

Exemplo com Diferentes Estilos de Marcadores

A propriedade CSS list-style-type permite personalizar os marcadores:

Disc (Padrão)

        <ul style="list-style-type: disc;">
            <li>Café</li>
            <li>Chá</li>
            <li>Leite</li>
        </ul>

Círculo

        <ul style="list-style-type: circle;">
            <li>Café</li>
            <li>Chá</li>
            <li>Leite</li>
        </ul>

Quadrado

        <ul style="list-style-type: square;">
            <li>Café</li>
            <li>Chá</li>
            <li>Leite</li>
        </ul>

Nenhum Marcador

        <ul style="list-style-type: none;">
            <li>Café</li>
            <li>Chá</li>
            <li>Leite</li>
        </ul>

Exemplo de Listas Aninhadas

Uma lista que contém outra lista:

        <ul>
            <li>Café</li>
            <li>Chá
                <ul>
                    <li>Chá preto</li>
                    <li>Chá verde</li>
                </ul>
            </li>
            <li>Leite</li>
        </ul>

Exemplo de Menu Horizontal

Estilizando uma lista para funcionar como um menu de navegação:

        <ul style="list-style-type: none; margin: 0; padding: 0; overflow: hidden; background-color: #333;">
            <li style="float: left;"><a href="#home">Home</a></li>
            <li style="float: left;"><a href="#news">Notícias</a></li>
            <li style="float: left;"><a href="#contact">Contato</a></li>
            <li style="float: left;"><a href="#about">Sobre</a></li>
        </ul>
