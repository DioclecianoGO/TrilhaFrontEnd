Entidades HTML

As entidades HTML permitem adicionar símbolos ou letras que não estão disponíveis no teclado a uma página HTML.

Entidades de Símbolos HTML

Para inserir símbolos que não podem ser digitados diretamente, você pode usar o nome da entidade ou um número de entidade (decimal ou hexadecimal).

<p>Eu vou exibir €</p>

Esse código exibe o símbolo do euro:

€

Outros exemplos incluem:

  • € para exibir:
    €
  • € para exibir:
    €
