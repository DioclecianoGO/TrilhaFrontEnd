Trabalhando com Iframes no HTML

O elemento <iframe> permite incorporar conteúdo externo dentro de uma página web. Ele utiliza o atributo src para definir o URL da página a ser exibida e o atributo title para acessibilidade.

Iframe Básico

Você pode ajustar a altura e a largura de um iframe usando os atributos height e width:

<iframe src="https://www.example.com" height="200" width="300" title="Iframe Exemplo"></iframe>
