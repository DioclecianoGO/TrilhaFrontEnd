Favicon e Ícones em HTML

Um favicon é uma pequena imagem exibida ao lado do título da página na aba do navegador. Essa imagem ajuda a identificar rapidamente a página entre várias abertas.

Como Adicionar um Favicon em HTML

Para adicionar um favicon ao seu site, você pode usar qualquer imagem. É recomendável que o favicon seja uma imagem simples e de alto contraste, pois isso facilita a visualização.

Você deve salvar a imagem do favicon no diretório raiz do servidor ou em uma pasta chamada images. Um nome comum para um favicon é favicon.ico.

Para adicionar o favicon no seu arquivo index.html, insira o seguinte código após o elemento <title>:

<link rel="icon" type="image/x-icon" href="/images/favicon.ico">

Exemplo de Código HTML com Favicon

O código a seguir ilustra como configurar o favicon:


<!DOCTYPE html>
<html>
<head>
    <title>Meu Título de Página</title>
    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>
<body>

    <h1>Este é um Cabeçalho</h1>
    <p>Este é um parágrafo.</p>

</body>
</html>

Atributos do Elemento <link>

Os atributos rel e type do elemento <link> desempenham funções importantes na definição de um favicon:

Atributo rel

O atributo rel especifica a relação entre o documento HTML e o recurso vinculado. Para favicons, o valor mais comum é "icon", que informa ao navegador que o recurso deve ser tratado como um ícone da página.

Atributo type

O atributo type indica o tipo de mídia do recurso. Para favicons, o valor típico é "image/x-icon", que informa ao navegador que o arquivo é um ícone no formato .ico.

Resumo

Em resumo, o uso correto dos atributos rel e type é essencial para garantir que o favicon seja reconhecido e exibido adequadamente pelos navegadores, contribuindo para a identidade visual do site.
