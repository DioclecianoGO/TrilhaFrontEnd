Tamanhos de Tabelas HTML

O conceito de Tamanhos de Tabelas HTML permite que desenvolvedores ajustem a largura e altura de tabelas, colunas e linhas, garantindo uma apresentação clara e organizada dos dados.

Largura da Tabela

Para definir a largura de uma tabela, usa-se o atributo style no elemento <table>, permitindo que a tabela ocupe 100% da largura disponível do elemento pai. Isso é ideal para layouts responsivos.

<table style="width:100%">
    <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
    </tr>
</table>

Largura de Colunas Específicas

A largura de colunas específicas pode ser ajustada em elementos <th> ou <td>. Por exemplo, uma coluna pode ser configurada com 70% de largura.

<table style="width:100%">
    <tr>
        <th style="width:70%">Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
    </tr>
</table>

Altura das Linhas

A altura das linhas também é manipulável, utilizando o atributo style no elemento <tr>. Isso possibilita criar linhas com alturas fixas, como 200px, para realçar determinadas informações.

<table style="width:100%">
    <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
    </tr>
    <tr style="height:200px">
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
    </tr>
</table>

Essas propriedades proporcionam aos desenvolvedores controle detalhado sobre o layout das tabelas, contribuindo para interfaces mais estéticas e funcionais.
