HTML Ordered Lists

O elemento <ol> em HTML é utilizado para definir uma lista ordenada, que pode ser numérica ou alfabética. Para iniciar uma lista ordenada, utiliza-se a tag <ol>, e cada item da lista é iniciado com a tag <li>. Por padrão, os itens da lista são numerados sequencialmente.

Atributo Type

A tag <ol> possui um atributo chamado type, que permite definir o estilo do marcador dos itens da lista. Os valores possíveis para este atributo incluem:

  • type="1": numeração com números (padrão).
  • type="A": numeração com letras maiúsculas.
  • type="a": numeração com letras minúsculas.
  • type="I": numeração com números romanos maiúsculos.
  • type="i": numeração com números romanos minúsculos.

Exemplos de Listas Ordenadas

Numeração com Números

<ol type="1">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Numeração com Letras Maiúsculas

<ol type="A">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Numeração com Letras Minúsculas

<ol type="a">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Numeração com Números Romanos Maiúsculos

<ol type="I">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Numeração com Números Romanos Minúsculos

<ol type="i">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Controlando a Contagem

É possível controlar a contagem da lista utilizando o atributo start, que define o número inicial da contagem. Por padrão, as listas começam a contagem a partir de 1, mas é possível iniciar a contagem a partir de um número específico. 

<ol start="50">
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ol>

Listas Aninhadas

As listas também podem ser aninhadas, ou seja, é possível incluir uma nova lista dentro de um item da lista existente. A tag <li> pode conter não apenas outra lista, mas também outros elementos HTML, como imagens e links, permitindo maior flexibilidade na estruturação do conteúdo. 

<ol>
  <li>Café</li>
  <li>Chá
    <ol>
      <li>Chá preto</li>
      <li>Chá verde</li>
    </ol>
  </li>
  <li>Leite</li>
</ol>
