HTML Links - Explorações e Exemplos

Este documento explora o uso de links em HTML, como funcionam, como podemos personalizá-los e exemplos práticos para aprender mais sobre o assunto.

O que é um Link em HTML?

Um link (hiperlink) em HTML é usado para criar uma conexão entre uma página web e outra. Quando você clica em um link, você é redirecionado para outro documento. Veja o exemplo abaixo de um link simples:

Visite o W3Schools!

Ao clicar, a página será aberta em uma nova aba, pois utilizamos o atributo target="_blank".

Links Relativos e Absolutos

Há dois tipos de URLs que você pode usar em um link: absolutos e relativos. URLs absolutos são endereços completos, como https://www.google.com. URLs relativos apontam para recursos no mesmo site.
